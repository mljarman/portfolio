Toxicity Classifier

Team Project

8-week project with 4 Data Science students for an outside stakeholder. We were tasked with creating an API that would be able to analyze text using natural language processing to detect varying degrees of toxicity.

Extensive research and testing was done on various machine learning models. The initial logistic regression with TF-IDF vectorizer was hard to beat but eventually was by a neural network which utilized FastText.

Our main goal was to focus on precision and recall scores, as false negatives could prove to be catastrophic. My role was to test out different models (including the hard to beat logistic regression with TF-IDF) as well as creating the API and predictive endpoint with a teammate.
