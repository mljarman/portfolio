Toxicity Classifier

Team Project

8-week project with 4 Data Science students for an outside stakeholder. We were tasked with creating an API that would be able to analyze text using natural language processing to detect varying degrees of toxicity.

Extensive research and testing was done on various machine learning models. The initial logistic regression with TF-IDF vectorizer was hard to beat but eventually was by a neural network which utilized FastText. The only problem was the size and memory needed to deploy the model onto Google, it proved to be expensive to run.

Our main goal was to focus on precision and recall scores, as false negatives could prove to be catastrophic in use cases, which included schools and professional development platforms. My role was to test out different models (including the hard to beat logistic regression with TF-IDF) as well as creating the API and predictive endpoint with a teammate.

This was my most passionate project I worked on and could see how technology really can be a tool used for good, and not just technology for technology's sake.
