Opti

Team Project

Completed in 1-week on a crossfunctional team of 7 including data science and web developer students. App designed to enable users to post and save Airbnb listings in Berlin at an optimal price using predictive modeling.

As the Data Engineer on the project, I was responsible for the API that connected the Machine Learning Engineer's predictive model to the backend engineer's database. The user would input various features like square footage, rooms, and beds and the API would take that information, in addition to a description given. Before going through the model, the description portion was transformed into word vectors using the Spacy embeddings. After going through the predictive model, the API returned the optimal price and was stored for the user in their listings.
