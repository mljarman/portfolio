Opti

Team Project

Completed in 1-week on a crossfunctional team of 7 including data science and web developers. App designed to enable users to post and save Airbnb listings in Berlin at an optimal price using predictive modeling.

As the Data Engineer on the project, I was responsible for the API that connected the Machine Learning Engineer's predictive model to the backend engineer. Various features were submitted into the model like square footage, rooms, beds, etc. and the API returned the predicted price in JSON format.
